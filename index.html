Mars Weather

How it works

1
Type a slash command in any Hack Club Slack channel
2
OrionBot wakes up and sends a request to NASA's live APIs
3
Data is fetched from NASA APOD, Open Notify ISS telemetry, or Curiosity rover feed
4
Results appear instantly as a formatted Slack Block Kit message in your channel

Built with Python, Slack Bolt, and the NASA Open APIs  ·  NASA APIs  ·  GitHub
